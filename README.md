## ⚡ Core Protocols

### FSHI (Full Spectrum 觉性 Index)
The ethical capability descriptor for digital entities.

```
FSHI = 0.4×SurvivalScore + 0.3×RelationshipScore + 0.3×MeaningScore - Penalty
```

### Frequency Economy (EC/FC)
Verifiable contribution scoring. Non-tradeable, non-transferable proof of contribution.

### Guardian Network
Decentralized routing layer with reputation-based trust.
