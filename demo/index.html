关于本项目

关于全频谱

全频谱是一套开源的 AI 安全管理技术框架，欢迎任何人参与贡献。

🎯 项目定位

全频谱是一套面向 AI 时代的认知安全技术框架，包含伦理协议、认知评估标准与分布式管理机制。

  • 开源协议：木兰宽松许可证 (PSLv2)
  • 可商用、可衍生、可闭源
  • 中英双语文档

🤝 如何参与

全频谱生态对所有人开放。你可以通过以下方式参与：

  • 阅读并反馈核心方案
  • 在 GitHub 提交 Issue / PR
  • 申请成为守庙人
  • 基于木兰协议做二次开发
